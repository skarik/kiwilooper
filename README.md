ORBITAL
=======

A scifi horror RPG.

Originally a [LD47 submission](https://ldjam.com/events/ludum-dare/47/tally-marks), now a larger project!

Follow along with development on the [Trello](https://trello.com/b/pHZEHOZg/tallymarks-development).

## Dev Prerequisites

GameMaker 2022 or later. Must be built in 64-bit mode.

## Dev stuff used

* XPanda : https://github.com/GameMakerDiscord/Xpanda
* CE : https://github.com/blueburncz/CoreExtension
